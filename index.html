Rally ALM Integration

Important Note for 0.1 Users

The installation instructions have changed. Changes will come down automatically per the plugin update system, rather manually installing from disk. Uninstall your 0.1 install and pull down the plugin from a custom repository, per instructions below.

Instructions

  • Add a custom plugin repository, per the instructions provided by Jetbrains
    • Configure the repository url to:
      http://rallysoftware.github.io/intellij-plugin/updatePlugins.xml
  • Click "Browse Repositories" and add the the rally-intellij plugin
  • Restart IntelliJ
  • Configure Preferences -> Rally

Issues

Note: Mis-configuring the Rally section or failing to configure it before use might result in undesirable behavior, as not all exception conditions are well handled.

Please visit the Github issues page for a list of known limitations and defects or to submit defects.
